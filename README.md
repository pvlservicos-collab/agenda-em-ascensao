# Agenda em Ascensão

Aplicativo web de agenda (single-page, HTML/CSS/JS puro) — "Minha Agenda em Ascensão".

## Como usar

Basta abrir o arquivo [index.html](index.html) diretamente no navegador. Não há build nem dependências.

## Publicar com GitHub Pages (opcional)

No repositório, vá em **Settings → Pages**, selecione a branch `main` e a pasta `/ (root)`. O site ficará disponível em `https://pvlservicos-collab.github.io/agenda-em-ascensao/`.
